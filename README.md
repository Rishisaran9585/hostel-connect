# Coimbatore Hostel Owner Association (CHOA)

This is the official website for the Coimbatore Hostel Owner Association (CHOA).

## Technologies Used

- Vite
- React
- TypeScript
- Tailwind CSS
- Shadcn UI
- Lucide React

## Development

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start dev server:
   ```sh
   npm run dev
   ```

3. Build for production:
   ```sh
   npm run build
   ```
